== Description ==

Wordpress to Steemit is simple way to sync your posts and pages to steemit.

This plugin offers the following features:

= Sync Wordpress posts and pages =
Once your posts or pages are published, they are also ready to show on steemit community.

== Installation ==

There are 2 easy ways to install our plugin:

=1.The standard way=
* In your WordPress Admin, go to menu Plugins > Add
* Search for "Wordpress to Steemit"
* Click to install
* Activate the plugin
* If your current user can edit pages or post, you can see "Steemit" button in any posts and pages you published

=2.The nerdy way=
* Download the plugin (.zip file) on the right column of this page
* In your WordPress Admin, go to menu Plugins > Add
* Select the tab "Upload"
* Upload the .zip file you just downloaded
* Activate the plugin
* If your current user can edit pages or post, you can see "Steemit" button in any posts and pages you published.
Your wordpress content will be automatically converted from html to markdown which supported by steemit
